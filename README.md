# vue-mars3d

## 首次运行前安装依赖
```
npm install
```

### http运行项目
```
npm run serve
```

### 打包编译项目
```
npm run build
```

### Lints和修复文件
```
npm run lint
```

### 自定义配置
配置引用(https://cli.vuejs.org/config/).

### 本项目出自于
官方github(https://github.com/marsgis/MarsGIS-for-Cesium).

### 注意事项

建议使用vue-cli@3以下的版本，经过测试目前vue-cli4@代理配置无效。
